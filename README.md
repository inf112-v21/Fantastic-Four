# INF112 RoboRally

This project uses:
*   Java 15.02
*   JUnit 4.11
*   Maven

To run:
  - Clone to your local machine 
  `git@github.com:inf112-v21/Fantastic-Four.git`
  - Compile using Maven 
  - Open project in your favorite IDE or CLI
      - IDE
        - In Import project to IntelliJ (or suitable alternative)
        -  Run `Main.java`
